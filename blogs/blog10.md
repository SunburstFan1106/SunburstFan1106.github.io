# 高中物理必修一

## 质点

### 定义

- 忽略了形状和大小，只有**质量**的点。
- **理想模型法**

### 可以将物体看作质点的条件

物体的**形状和大小**对研究问题没有影响，如：

*车的尺寸远小于路程、公转*

## 参考系

### 定义

- 为了研究物体运动而选取的**假定静止**的物体。

参考系不是绝对静止，是人为的假定静止。参考系可以任意选取，默认是**大地**。

选取不同的参考系，物体的运动状态不同。**谁不动，谁就是参考系。**

### 坐标系

是参考系的一种，注意**需要带单位。**

 ## 时间与时刻

### 定义

- **时刻**：一个瞬间，时间轴上的一个点。
- **时间（间隔）**：一段时间，时间轴上的一段。

### 常见说法

- **时刻**：$0$ 时刻，第 $1$ 秒初，第 $1$ 秒末，第 $n$ 秒初 ...

- **时间**：第 $1$ 秒，第 $n$ 秒，前 $n$ 秒。

> [!warning]
>
> 虽然[电流](https://zh.wikipedia.org/wiki/電流)具有方向性（由正极流向负极），但由于不符合矢量运算，所以不会是矢量  。
>
> 来自[维基百科](https://zh.wikipedia.org/zh-cn/%E6%A0%87%E9%87%8F_(%E7%89%A9%E7%90%86%E5%AD%A6)#:~:text=%E4%B8%8E%E4%B9%8B%E7%9B%B8%E5%AF%B9%EF%BC%8C%E5%90%91%E9%87%8F%EF%BC%88%E5%8F%88,%E4%B8%8D%E5%90%8C%E7%9A%84%E5%80%BC%EF%BC%8C%E4%BE%8B%E5%A6%82%E9%80%9F%E5%BA%A6%E3%80%82)。

## 矢量与标量

### 定义

- **矢量**：有方向和大小的物理量。例如，*速度、力、位移*。
- **标量**：只有大小的物理量。例如，*温度、质量、时间*。

### 一维矢量的代数表示

数字+正负号。

> [!caution]
>
> 有些标量也可以用数字+正负号表示。

## 位移与路程

### 定义

- **位移**：物体位置变化的量，指从起点到终点的直线距离和方向。
- **路程**：物体运动轨迹的长度。

### 根据坐标计算位移

末位置坐标-初位置坐标

### 二者关系

路程 $\geq$ 位移大小，等号成立，当且仅当**单向直线运动**。

## 速度

### 定义

#### 平均速度

物体单位时间内的**位移**，表示物体位置变化的**快慢和方向**。

平均速度是一个**矢量**，方向是位移的方向。

- **公式**：$v=\frac {x}{t}$，其中 $x$ 表示位移（矢量）。

- **单位**：$m/s$。

计算平均速度不管物体运动的中间过程，只看结果。

> [!NOTE]
>
> 速度一定要带方向。
>
> 平均速度记录的是**一段时间**的事情，不是一个时刻。
>
> 平均速度是记录结果的，无法反映运动的过程。
>
> 题目中所出现的速度，都理解为**瞬时速度**即可。

#### 平均速率

物体单位时间内的**路程**，表示物体**运动的快慢**（定义与初中的*速度*类似）。

平均速率是一个**标量**。

- **公式** $v=\frac{s}{t}$，其中 $s$ 表示路程（标量）。

- **单位**：$m/s$。

> [!caution]
>
> 平均速率不是平均速度的大小，需要和*瞬时速度&瞬时速率*区分开。
>
> 特殊地，单项直线运动时，平均速度大小 $=$ 平均速率。

#### 瞬时速度

物体在某一时刻或位置的**速度**，是平均速度在时间间隔趋于 $0$ 时的极限。

是一个**矢量**，方向是物体运动轨迹的切线。

**匀速直线运动**的定义：**瞬时速度**不变的运动。

#### 瞬时速率

**瞬时速度的大小**，表示物体在**某一时刻**的运动快慢。

### 大小关系

平均速率 $\geq$ 平均速度的大小；瞬时速率 $=$ 瞬时速度的大小（定义）。

> [!IMPORTANT]
>
> 当时间极短时，一律当成匀速直线运动。

## 加速度

### 速度变化量

#### 变化量

表示一个物理量在初始状态和最终状态之间的差值。

$\Delta x= x_{final}-x_{initial}$，其中 $x_{final}$ 和 $x_{initial}$ 分别表示物理量 $x$ 的最终和初始状态。

##### 标量变化量

如果物理量是标量，则变化量也是标量。例如，*温度变化量、质量变化量*等。

正负表示**增减**，数字表示**多少**。

> [!warning]
>
> **变化量和初、末状态的关系**
>
> 变化量和初状态是独立的，彼此无关。
>
> 末状态与初状态和变化量均有关。
>
> 变化量又外界因素决定。与初状态、末状态均无关。

##### 矢量变化量

如果物理量是矢量，则变化量也是矢量。例如，*位移变化量*、*速度变化量*等。

**图示表示**：

矢量变化量可以用带有箭头的线段来表示，箭头的方向表示变化量的方向，长度表示变化量的大小。

箭头与初始状态同向表示增加，反向表示减少。

#### 直线运动速度变化量

速度变化量显然属于矢量变化量，除了可以用箭头表示之外，还可以用正负号加数字表示：

- 规定一个正方向。
- 确定初速度 $v_{initial}$ 和末速度 $v_{final}$ 的正负。
- 将正负号当成数学上的正负，计算 $v_{final}-v_{initial}$。

> [!note]
>
> 一般以初速度方向为正方向。

### 变化率

单位时间内的变化量，表示变化的快慢。

$变化率=\frac{变化量}{时间间隔}$。

> [!important]
>
> 变化量表示变化的多少，变化率表示变化的快慢。
>
> *速度是位移的变化率。*

### 速度变化率（加速度）

表示速度变化快慢的物理量。

- **定义式**：$a=\frac{\Delta v}{\Delta t}$，注意这个定义式和上文的变化率定义式一致。

- **单位**：$m/s^2$。

速度变化率是一个矢量，其方向与 $\Delta v$ 一致。

### 计算直线运动的加速度

1. 规定正方向。

2. 计算 $\Delta v$。

3. 根据定义式计算加速度。

# [COCI 2024/2025 #2] 差异

## 题目大意

给定三个正整数 $n，m，k$，以及两个周期为 $n,m$ 的无线整数序列 $a$ 和 $b$。构造无限序列 $c$，求： $\sum _{i=1}^{k} c_i=a_i \oplus b_i$

## 解题思路

读题可以发现 $c$ 的循环节长度为 $\text lcm(n,m)$，那么就有一个 $O(\text lcm(n,m))$ 的暴力，可以获得 57pts。

考虑优化这个暴力，不难想到拆位计算贡献，对于每一个二进制位，先求出在周期内的贡献，再求出周期外的散段的贡献。

可以手模一下计算 $c$ 的过程，不难发现，对于序列 $a$ 中的元素 $a_i$，第一个与其进行异或运算的序列 $b$ 元素一定是 $b_i$，第二个则是 $b_{(i+n) \mod m}$，以此类推。

如果将这些元素全部一一建边，最终一定会形成一个环，且环的长度是 $\frac {m}{\gcd(n,m)}$。

考虑将这些环一个个存下来，然后跑一下环上前缀和，就可以开始统计贡献：

计算出周期的个数 $c$ 和剩余长度 $rst$。

- 如果 $a_i=0$，则对应的 $b$ 序列中该位的贡献就是整个环的总和乘 $c$，$rst$ 的贡献可以通过前缀和相减得到。

- 如果 $a_i=1$，则产生的贡献就是 $b$ 序列中该位 $0$ 的个数，即将总环长减去环的总和乘 $c$，$rst$ 的贡献就是拿 $rst$ 减去这一段的和得到。

统计完一位的贡献后乘上 $2^{bit}$，$bit$ 为当前位数，全部加起来就是答案。

**核心代码**：

这里给出计算每一位贡献的函数 `calc`。

```cpp
vector<int> p(2e5+5,0);
int calc(vector<int> &a,vector<int> &b){
    vector<vector<int> >cir(2e5+5,vector<int>(1));
    
    int GCD=__gcd(n,m),t=m/GCD;
    for(int i=0;i<GCD;i++){
        int head=i;
        for(int j=0;j<t;j++){
            p[head]=j+1; //记录该位对应的编号
            cir[i].push_back(b[head]); //记录这个环
            head=(head+n)%m; //更新指针
        }
        for(int j=0;j<t;j++){
            cir[i].push_back(cir[i][j+1]); //将环复制一遍，方便跑前缀和
        }
        for(int j=1;j<cir[i].size();j++){
            cir[i][j]=(cir[i][j]+cir[i][j-1])%mod; //计算前缀和
        }
    }

    int res=0;
    for(int i=0;i<n;i++){
        if(i>k)break;

        int x=p[i%m]; //找到编号
        int c=(k-i)/(n*t)%mod,rst=(k-i)%(n*t);
        // 算出 c 和 rst
        rst=(rst+n-1)/n; //向上取整
        if(a[i]==0){ //计算贡献部分，可以对照解题思路
            res=(res+c*cir[i%GCD][t]%mod)%mod;
            res=(res-cir[i%GCD][x-1]+cir[i%GCD][x+rst-1]+mod)%mod;
        }
        else{
            res=(res+c*(t-cir[i%GCD][t])%mod)%mod;
            res=(res+rst+cir[i%GCD][x-1]-cir[i%GCD][x+rst-1]+mod)%mod;
        }
    }

    return res;
}
```
